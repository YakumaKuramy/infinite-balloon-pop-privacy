# Política de Privacidade - Infinite Balloon Pop

Última atualização: Junho de 2026

A Mandacaru Indie Game Dev desenvolveu o jogo **Infinite Balloon Pop** como um aplicativo gratuito (Free / Ad-Free). Este serviço é fornecido sem custo e foi projetado para uso no estado em que se encontra.

### 1. Coleta e Uso de Informações
Nós não coletamos, armazenamos ou compartilhamos qualquer tipo de informação pessoal identificável dos nossos usuários. O jogo funciona de forma totalmente offline e não possui sistemas de cadastro, login ou rastreamento de dados de localização.

### 2. Dados Técnicos e Estatísticas
O aplicativo não utiliza serviços de terceiros que possam coletar informações usadas para identificar você. Dados de progresso do jogo (como sua pontuação máxima) são salvos exclusivamente de forma local no armazenamento interno do seu próprio dispositivo e são apagados caso o jogo seja desinstalado.

### 3. Alterações nesta Política de Privacidade
Podemos atualizar nossa Política de Privacidade de tempos em tempos. Portanto, é aconselhável revisar esta página periodicamente para verificar quaisquer alterações. Avisaremos sobre quaisquer mudanças publicando a nova Política de Privacidade nesta página.

### 4. Contato
Se você tiver alguma dúvida ou sugestão sobre nossa Política de Privacidade, não hesite em entrar em contato conosco pelo e-mail do desenvolvedor cadastrado na Google Play Store.
